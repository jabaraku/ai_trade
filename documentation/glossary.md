# Glossary

## OHLCV

Open, High, Low, Close, Volume. The standard fields for a price bar.

## Price bar

A row representing market activity over a time interval, such as one day, one week, or one month.

## Candlestick

A visual representation of open, high, low, and close for one period.

## Bullish candle

A candle where close is above open. The dashboard colors these green.

## Bearish candle

A candle where close is below open. The dashboard colors these red.

## Feature

A numeric input derived from data. Example: RSI, ATR, return, volume ratio.

## Target

A future-looking label used for supervised machine learning. Example: whether the next 5-day return is positive.

## Look-ahead bias

A modeling error where future information leaks into training features. It can make a strategy look much better in testing than it would have been in real life.

## XGBoost

A gradient-boosted tree machine learning library that is strong for tabular data.

## Gemma

An open-weight language model family from Google. In this project, it is used locally through Ollama for explanation.

## Ollama

A local runtime for downloading and running LLMs on your machine.

## DuckDB

A local analytical database used here to store price data.

## yfinance

A Python package used to download Yahoo Finance data for prototyping and research.

## RAG

Retrieval-Augmented Generation. A design where an LLM receives relevant retrieved documents at question time instead of relying only on its model weights.

## ATR

Average True Range. A volatility indicator based on true range.

## RSI

Relative Strength Index. A momentum oscillator that compares recent gains with recent losses.

## MACD

Moving Average Convergence Divergence. A momentum indicator based on the difference between two exponential moving averages.
